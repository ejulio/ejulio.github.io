Recursos

2016/06/08

Coleção de links que considero interessantes e que podem servir para futuras referências. Alguns links de repositórios e pesquisadores também podem ser encontrados no GitHub.

Frameworks / Libraries

  • OpenCV [python, C++]: Biblioteca para visão computacional e aprendizado de máquina.
  • Dlib [python, C++]: Biblioteca para processamento de imagens, aprendizado de máquina e outros.
  • 4DFace [C++]: 3DMM para construir um modelo 3D face a partir de vídeos 2D.
  • OpenFace [C++]: Ferramenta para análise de comportamento facial.
  • Menpo [python]: Framework com métodos para modelagem deformável 2D e 3D.
  • VLFeat [C, MATLAB]: Biblioteca para visão computacional.
  • SimpleCV [python]: Biblioteca para visão computacional.
  • NLTK [python]: Ferramenta processamento de linguagem natural.
  • scikit-image [python]: Biblioteca para processamento de imagens.
  • scikit-learn [python]: Biblioteca para aprendizado de máquina.
  • Mahotas [python]: Biblioteca para visão computacional.
  • clmtrackr [javascript]: Biblioteca para tracking de características faciais usando Contrained Local Models.
  • Accord.NET [C#]: Framework para aprendizado de máquina e visão computacional.

Datasets

  • Head Pose Database: Database para estimativa da pose da cabeça (pitch, yaw e roll).
  • Head Pose Image Database: Database para estimativa da pose da cabeça (pitch, yaw).
  • Biwi Kinect Head Pose Database: Database para estimativa da pose da cabeça (pitch, yaw).
  • Annotated Faces in the wild - AFW: Database para detecção de faces e landmarks em ambientes não controlados.
  • AFLW2000-3D: Database para alinhamento de faces com rotação.

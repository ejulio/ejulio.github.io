Frameworks / Libraries

  • OpenCV [python, C++]: Biblioteca para visão computacional e aprendizado de máquina.
  • Dlib [python, C++]: Biblioteca para processamento de imagens, aprendizado de máquina e outros.
  • OpenFace [C++]: Ferramenta para análise de comportamento facial.
  • VLFeat [python]: Biblioteca para visão computacional.
  • SimpleCV [python]: Biblioteca para visão computacional.
  • NLTK [python]: Ferramenta processamento de linguagem natural.
  • scikit-image [python]: Biblioteca para processamento de imagens.
  • scikit-learn [python]: Biblioteca para aprendizado de máquina.
  • Mahotas [python]: Biblioteca para visão computacional.
  • clmtrackr [javascript]: Biblioteca para tracking de características faciais usando Contrained Local Models.
  • Accord.NET [C#]: Framework para aprendizado de máquina e visão computacional.
